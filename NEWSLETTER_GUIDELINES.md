# Newsletter Structure Guidelines

## Four-Section Format

1. **From Steven, Founder and CEO** - Personal update from founder with milestones and vision
2. **What to expect in Shanghai** - Monthly local insights, hidden gems, and travel tips
3. **Featured Host** - Spotlight on one host with profile and cuisine description
4. **Help us Grow** - Call-to-action for forwarding and recommendations

## Mobile-First Design Principles

- **Keep it concise** - Short paragraphs optimized for mobile reading
- **Break lines frequently** - Form new paragraphs for easy scanning
- **Maximum one photo per section** - Avoid overwhelming mobile viewers
- **Clear section headers** - Help readers navigate quickly

## Featured Host Format

- Start with **name and cuisine** (no title like "Featured Host")
- Include **profile photo only** (no food photos)
- Structure: Name → Cuisine → Description → Philosophy (as separate paragraph)

## Pinyin Formatting

- Use **dashes between syllables** for non-Chinese speakers
- Examples: Wu-Kang Road (not Wu Kang Road), Pan-Long-Xin-Tian-Di (not Pan Long Xin Tian Di)
- Include P.S. explaining this formatting choice for reader education
